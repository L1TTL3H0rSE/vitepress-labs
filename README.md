Этот проект представляет собой сборник лабораторных работ по ключевым дисциплинам Computer Science, переосмысленный в формате интерактивного веб-сайта. Вместо статичных листингов кода, каждая работа сопровождается живыми демонстрациями, эмуляторами и песочницами, позволяющими тестировать алгоритмы прямо в браузере.

## 🚀 Ключевые Особенности

* **Собственный Язык Программирования:** Разработан компилятор для простого русскоязычного языка, который включает в себя:
  * **Лексический анализатор (Лексер):** Разбивает код на токены.
  * **Синтаксический анализатор (Парсер):** Строит Абстрактное Синтаксическое Дерево (AST) методом рекурсивного спуска.
  * **Транслятор (Кодогенератор):** Преобразует AST в исполняемый JavaScript-код.
* **Эмулятор Машины Тьюринга:** Интерактивный компонент для визуализации работы машины Тьюринга с программами сложения, вычитания и копирования в унарной системе.
* **Визуализатор Конечных Автоматов:** Реализован алгоритм детерминизации НКА в ДКА (метод подмножеств) с визуализацией графов при помощи Mermaid.js.
* **Интерактивные Инспекторы Структур Данных:** Специальные Vue-компоненты (`ClassRunner`) позволяют в реальном времени взаимодействовать с реализованными структурами данных (стеки, очереди, деревья), вызывая их методы и наблюдая за изменением состояния.
* **Мобильное Приложение по Информационной Безопасности:** Демонстрация принципов ИБ на Flutter, включая симметричные шифры, OAuth 2.0 (VK ID), биометрическую аутентификацию и безопасное хранение паролей с использованием **Bcrypt**.

## 🛠️ Технологический Стек

* **Frontend & Документация:** [VitePress](https://vitepress.dev/), [Vue 3](https://vuejs.org/) (Composition API), [TypeScript](https://www.typescriptlang.org/), SASS.
* **Алгоритмы и Структуры Данных:** TypeScript, Python.
* **Мобильная разработка (Инфобез):** [Flutter](https://flutter.dev/), Dart.
* **CI/CD:** GitHub Actions для автоматического деплоя на GitHub Pages.

## 🏛️ Архитектура

Основа проекта — сайт на VitePress, где стандартный Markdown расширяется кастомными Vue 3 компонентами. Это позволяет бесшовно встраивать сложную интерактивную логику в статическую документацию.

* **`InteractiveRunner.vue`:** Универсальный компонент для запуска функций с пользовательским вводом.
* **`ClassRunner.vue`:** Компонент, использующий рефлексию (прототипы) для динамического создания UI для любого переданного класса. Он автоматически находит публичные методы и генерирует для них поля ввода и кнопки.
* **`CompilerRunner.vue`:** Интегрированная среда, которая объединяет все этапы компиляции и запускает сгенерированный JS-код в изолированном окружении с помощью `new Function()`.

## 📂 Структура Проекта

```text

_vitepress-labs/
├── .github/workflows/      # CI/CD для деплоя на GitHub Pages
├── .vitepress/             # Конфигурация VitePress (тема, роутинг)
├── components/             # Глобальные Vue-компоненты (InteractiveRunner, ClassRunner и др.)
├── labs/                   # Исходный код и Markdown-файлы для каждой дисциплины
│   ├── discrete/
│   ├── infosec/            # Исходный код Flutter-приложения
│   ├── lingo/              # Код компилятора, автоматов, калькулятора
│   └── mathlogic/          # Реализации структур данных на TS и Python
└── public/                 # Статические ассеты (иконки, изображения)

```

## ⚙️ Локальная Установка

1. **Клонируйте репозиторий:**

    ```bash
    git clone https://github.com/L1TTL3H0rSE/vitepress-labs.git
    cd vitepress-labs
    ```

2. **Установите зависимости** (рекомендуется использовать `pnpm`):

    ```bash
    pnpm install
    ```

3. **Запустите dev-сервер:**

    ```bash
    pnpm docs:dev
    ```

    Сайт будет доступен по адресу `http://localhost:5173`.
