# Лабораторная работа №1. Графы и операции над ними.

**Цель:** Познакомиться с графами и базовыми операциями над ними. Изучение алгоритмов поиска в глубину и поиска компонент связности.

## Часть 1. Работа в Wolfram Mathematica

### Задание 1. Создание базовых графов

Нужно создать и изобразить:

- 2-регулярный граф с 5 вершинами (g1).
- Граф Турана с параметрами 5 и 3 (g2).

![Графы g1 и g2](/img/discrete/lab1/task1.png)

::: details Код решения

```mathematica
g1 = CycleGraph[5];
g2 = TuranGraph[5, 3];
{g1, g2}
```

:::

### Задание 2. Модификация графов

Создание производных графов:

- Дополнение `g3` для графа `g1`.
- Граф `g`, полученный соединением 1 и 3 вершины `g3`.
- Граф `g4` — звезда с 5 вершинами.

![Результаты задания 2](/img/discrete/lab1/task2.png)

::: details Код решения

```mathematica
g3 = GraphComplement[g1];
g = EdgeAdd[g3, 1 <-> 3];
g4 = StarGraph[5];
{g3, g, g4}
```

:::

### Задание 3. Операции над графами

Нахождение и визуализация результатов операций над `g1` и `g4`:

![Результаты задания 3](/img/discrete/lab1/task3.png)

::: details Общий код для всех операций

```mathematica
GraphJoin[g1, g4]
GraphComplement[g4]
GraphDifference[g1, GraphComplement[g4]]
GraphIntersection[g4, g1]
GraphProduct[g4, g1]
GraphSum[g4, g1]
GraphUnion[g4, g1]
```

:::

### Задание 4. Матрица смежности и подграфы

Создание графа `g5` из матрицы смежности и проверка его свойств.

1. **Матрица смежности:**

```mathematica
1 1 1 0
1 1 0 1
1 0 1 1
0 1 1 1
```

2. **Проверка:** Является ли граф псевдографом?
   *Ответ: Да (имеются петли, так как на диагонали есть единицы).*

![Результаты задания 4](/img/discrete/lab1/task4.png)

::: details Код решения

```mathematica
adj = {{1, 1, 1, 0}, {1, 1, 0, 1}, {1, 0, 1, 1}, {0, 1, 1, 1}};
g5 = AdjacencyGraph[adj];

(* Проверка на простоту *)
SimpleGraphQ[g5] 

(* Создание простого графа g6 *)
g6 = SimpleGraph[g5];

(* Объединение g5 и реберного графа g3 *)
GraphUnion[g5, LineGraph[g3]]

(* Подграф g7 с вершинами 1,2,3 *)
g7 = Subgraph[g5, {1, 2, 3}]
```

:::

**Формат вывода:**
